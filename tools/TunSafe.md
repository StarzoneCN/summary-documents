# <div style="text-align:center;color:#FF9900">TunSafe</div>

## 1. [Linux安装][linux-install]
### 1.1 拉取git
```sh
$ git clone https://github.com/TunSafe/TunSafe.git
$ cd TunSafe
```
### 1.2 安装编译工具
```sh
$ sudo apt-get install clang-6.0
```


### 1.3 编译
```sh
$ make
$ sudo make install
```
